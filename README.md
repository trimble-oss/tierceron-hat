The Wanderer
Welcome to the wanderer.  The 3 (or 4) commands (tiara, brimfeather, tipcap, and tipcaptwo) below illustrate
the activities of the wanderer.

To compile, run:
make tiara
make brimfeather
make captip
make captiptwo

To run:
./bin/tiara
./bin/brimfeather

./bin/tipcap
./bin/tipcaptwo # Optional
